A minimal Alpine Linux demo runing inside v86, a javascript x86 emulator.
  • Everything is running on your browser
  • Customized minimal kernel, not from Alpine's repo
  • Minimal setup with busybox
  • Upgradable but not persistent
  • Network support (not implemented)
  • Plan9 FS (not implemented)
  • Frame-buffer (not implemented)
  • Serial (not implemented)
  • Mouse (not implemented)
